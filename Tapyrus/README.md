# Tapyrus

TapyrusはChaintope社が開発するBitcoinフォークのコンソーシアム型ブロックチェーンです。複数の署名者で形成するSignerネットワークによる検証をベースにブロックが進行する「Proof of Authority」を採用していることが特徴です。
